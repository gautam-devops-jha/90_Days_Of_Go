# Arithmetic Operations

Go has multiple Airthmetic operators
`a,b := 10,5` // Go allows multiple variables to be initialized at once! 

1. **`+` :-** Addition `c := a+b`  
2. **`-` :-** Subtraction `c = a - b`
3. **`*` :-** Multiplication `c = a * b`
4. **`/` :-** Division `c = a / b`
5. **`%` :-** Modulas `c = a % b` Gives reminder
